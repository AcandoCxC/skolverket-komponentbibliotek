Laddningsindikator

Använd den här komponenten för att indikera någon slags asynkron aktivitet som påverkar användaren.

För utvecklare

Både spinnern och throbbern går att skala med CSS. Använd då width, så behåller den underliggande SVGn sin aspektratio.

  • Content: 
    .skv-spinner {
  width: 100px;
  stroke-linecap: round;
}

.skv-spinner #spinner__background {
  fill: #99CED4;
}

.skv-spinner #spinner__arm, .skv-spinner #spinner__stroke {
  stroke: #262626;
  stroke-width: 5;
}

.skv-spinner #spinner__arm {
  transform-origin: 50% 50%;
  animation: rotate 2s linear infinite;
}

.skv-spinner #spinner__stroke {
  transform: rotate(-90deg);
  transform-origin: 50% 50%;
  animation: dash 4s linear infinite;
}

@keyframes rotate {
  0% {
    transform: rotate(0deg);
  }
  100% {
    transform: rotate(360deg);
  }
}

$circumference: 80% * 3.14;

@keyframes dash {
  0% {
    stroke-dashoffset: 0%;
    stroke-dasharray: 0% $circumference;
  }
  50% {
    stroke-dashoffset: 0%;
    stroke-dasharray: $circumference 0%;
  }
  100% {
    stroke-dashoffset: - $circumference;
    stroke-dasharray: $circumference;
  }
}
