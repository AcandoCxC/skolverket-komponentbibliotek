SEO

Selector Description
title Defines the title of the current page. Read more ›

Progressive Web App

Selector Description
application.name Specifies the name of the Web application that the page represents.

Social data

We have focused on a broad selection of social options to enable content-rich share possibilities. Here are some key points to follow when configuring this section.

Selector Description
share.title Can choose a separate title for sharing. Page title will be used if share title is empty.
share.image Recommended size is 1200 x 630px. If an image is not available in this size, we should at least use 600 x 315px. Images must not exceed 8 MB. Read more ›
share.domain Describes the root url of the current page.
share.description Can choose a separate description for sharing. Meta description will be used if share description is empty.
share.image Provides an option to select a custom image per page when sharing, but should have a default (site-wide) fallback.
share.url The current page url
share.type Used for facebook og-tags. No other type than ‘website’ should be needed at this point.
share.twitterCardType Used to customise the appearance of the twitter card when sharing this page. summary: Simple card summary_large_image: Card with large image * player: Enables a media player. Requires share.video data.
share.video.width Defines the width of the video. This data should be retrieved programmatically.
share.video.height Defines the height of the video. This data should be retrieved programmatically.
share.video.src The url to the video.
share.video.type The mime-type of the video.
